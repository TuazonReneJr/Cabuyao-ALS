Folder Structure:

* index.php - Entry Point of Website
* assets/: Holds all static resources like CSS, JavaScript, images, fonts, and third-party libraries.
* includes/: Stores reusable components like headers, footers, modals, and toasts.
* pages/: Organized by user role (admin, teacher, student) and contains specific pages.
* php/: Contains server-side scripts, including database connections, form handling, and AJAX scripts.
* uploads/: Stores files uploaded by users, such as student documents and module files.
* data/: Contains JSON files for configurations and field mappings.
* logs/: Used for logging errors and access records.
* temp/: Temporary storage for processing files.
* docs/: Holds documentation related to the project.
